# War Amongst Heavens

## Features

- High difficulty, few enemies
- Randomized dungeons
- Slower, more strategic combat
  - Realtime turn-based (2s tick)
  - Action queue
  - Possible actions randomly drawn from a deck
- New world setting
- [[b13733a8]]-style currency

## Classes

- Warrior
- Alchemist
- Artisan
