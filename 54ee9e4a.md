# Programming Language

```
define[welcome][{
    [X, nil] -> format["Welcome {1}!", [X]]
    [X, Y]   -> format["Welcome {1} to {2}!", [X,Y]]
}]
# Is equivalent to
define[welcome, { X nil -> format["Welcome {1}!", [X]], X Y -> format["Welcome {1} to {2}!", [X,Y]] }]

# Similarly
sum[X,Y]
# Is equivalent to
sum[X][Y]
```
